sublimetext
===========

My SublimeText 2 setup

# Installed packages

- Package Control (https://sublime.wbond.net/)
- All Autocomplete
- GitGutter
- Pretty JSON
- Pretty YAML
- Pylinter (requires pylint to be installed: ``pip install pylint``)
- SideBarEnhancements
- SublimeCodeIntel (https://github.com/SublimeCodeIntel/SublimeCodeIntel)
